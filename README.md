## seven

人生とは何かを思い出すための教科書

### 何が好きなのかを探す

スティーブジョブズの言葉

`好きなことを見つけ、そのために人生を過ごしましょう。好きなことのためです。あとは何も考える必要ありません。`

### 目標は”したいことをする”

TODOリストをまず作ります。ただし、することリストではありません。したいことリストを作ります。これは、須藤元気さんの本で学んだことですが、することリストをあげても、人間することはできません。したくないことがたくさんありますから。なら、したいことリストを作ればいいのです。世界一周旅行をしたい、結婚したい、大企業に就職したい、スカイダイビングをしたい、など何でも構いません。そのあと、それをするためには何をすればいいのかを考えます。その考えたことを明日から実行するのです。そうすれば、あなたの人生の素晴らしいスタートが始まります。

### 日日是好日

いい日も悪い日もすべて良い日です。仕事でミスをした、告白した振られた、財布を落とした。確かに悪いことです。しかし、仕事でのミスは、次に生かせばいいです。私が大好きな言葉「大失敗をしたものだけが大成功をおさめることができる」ロバート・ケネディ　があります。告白して振られたのは、神様がこの相手と付き合うな、不幸になると言っている、財布を落としたのは、財布を落とさなかったらギャンブルで100万円使っていた、などと考えるのです。すべてラッキーなのです。良い日なのです。

### 随所に主となれ

人生の主役は君だ。とカッコよいことを言いますが、本当にそうです。この宇宙、この世界、この日本のことを考えてください。誰のためにあるのでしょう。誰のために作られたのでしょう。答えはありません。あなたが生きているという事実だけは揺るぎありません。そのほかの人たちは、極端に言うと物体です。その物体の中には、あなたを助けてくれる、幸せにしてくれる良いものもあるかもしれません。ただ、邪魔な物体もたくさんあると思います。その物体に自分が主役の物語を邪魔されてはたまりません。そういうときは、あなたのなりたいように、したいようにしてください。この世界は終わろうとも、です。怖い話をしているわけではありません。本当にこの世界が何か解明されていない以上、主役はあなたです。周りに惑わされないで。

### 喫茶去

お茶を飲む時間はお茶を飲む、歩いているときは歩くのみ、お風呂のときはお風呂を楽しむ、そのときのことに向かい合うことはなかなかすることができない時代になりました。そのものと向き合うということが心の余裕、心の充実につながります。仕事中は仕事に向き合う、仕事を終えたあと、「今日は仕事集中していい仕事できたなあ」というのが増えるでしょう。それが幸せです。

### 本を読む

自分の人生を全うしてきた人たち、人生すべてをかけた知識をすぐに手に入れることができる。これほど、すごいことはないということを忘れてはならない。この本を読んでいるあなたは、著者が3年かけたこの本を一瞬で会得してしまったのです。おめでとうございます。

### 卒業する

１０代からしてきた趣味や楽しいことがあるでしょう。しかし、大人になるとできなくなることがあります。仕事を持つ、家庭を持つ、そのときに、その趣味に全力を注げないのであればやめてください。

### とにかくやってみる

とにかく失敗を恐れず挑戦してください。成功率0.0001%でもやってください。それをして成功した人が成功者です。失敗者、やってもいない人が世の中の99.9999%です。その周りにいる人たちに惑わされないでください。「できないに決まってる」、「無理だろ」、「バカじゃない」そういうのはごもっともです。しかし、0.0001%の確率があるならやってみましょう。そしてだめだったら、だめだったと笑えばいいのです。「だから言っただろう」となっても、「その通り」でよいのです。言葉で相手を否定するより、人生楽しんだ勝ちです。

### 学がない人へ

社会に出てから天才より１０倍楽しめ

学歴がないということは、頭の良い人とは脳の構造の時点で差があります。
その構造差は学生時代に差が大きく出てしまいます。
そのまま社会で競い合うと、負けてしまいます。
負けて悔しいのであれば、１０倍人生を楽しんでください。
楽しむためには努力も必要です。
社会に出てからは、勉強する時間が少なくなります。
社会という閉塞感、競争社会から能力が向上しない頭の良い方がほとんどです。
その中で努力し知識を失敗を重ね知識を蓄えていくことで、
頭の良い人より充実した人生を送ることができます。

### コミュニケーション能力とは何か

それは人に興味を持つこと

物体（人）に興味を持ちましょう。自分のためになる物体かを見極めるのです。自分のためになる物体だと確信したところで、その物体は人となります。ただ、自分にとって良い人は多くいません。多くは出会えませんので、慎重に見極めてください。
